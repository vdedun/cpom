Detailed Description

template<class T>
class cpom::OctreeNode< T >

Class defining an octree node that holds data of type T.

An octree is modeled by a tree of OctreeNode. Leaves contain a number of elements of type T. The octree can be traversed through visitor functions.

Example usage can be found in the unit test OctreeNode.ut.cpp.
